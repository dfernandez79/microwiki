Syntax Quick Reference

Paragraph
A paragraph is simply one or more consecutive lines of text, separated by one or more blank lines.
Headers
Headers are “underlined” using equal signs (for first-level headers) and dashes (for second-level headers). Example: 
This is an H1
============

This is an H2
-------------
Blockquotes
Uses email-style > characters for blockquoting. Example: 
> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Links
Write them like this: [This link](http://example.net/) , or using references: 
This is [an example] [id] reference-style link.

[id]: http://example.com/  "Optional Title Here"
Emphasis
Wrap text with * for bold or _ for italic. Example: *bold* or _italic_
Code
Use backtick quotes for inline code, ie: `printf()`. Start every line with 4 spaces or 1 tab for code blocks: 
This is a normal paragraph:

    This is a code block.
Images
Like links but with an exclamation mark (!) at the beginning. Example: ![My photo](/photo.jpg)
